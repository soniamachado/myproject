# Templates para Projeto Spring Boot

Esta pasta contém templates para criar novos arquivos rapidamente no projeto.

## Como usar templates no Cursor/VS Code

### Método 1: Usando Snippets (Recomendado)
1. Pressione `Ctrl+Shift+P` (ou `Cmd+Shift+P` no Mac)
2. Digite "Preferences: Configure User Snippets"
3. Selecione "java.json"
4. Adicione os snippets dos templates abaixo

### Método 2: Copiar e Colar
1. Abra o template desejado nesta pasta
2. Copie o conteúdo
3. Crie um novo arquivo Java
4. Cole e ajuste os nomes das classes/variáveis

### Método 3: Usando File Templates (IntelliJ IDEA)
Se estiver usando IntelliJ IDEA:
1. Clique com botão direito no pacote
2. Selecione "New" → "Java Class"
3. Ou use `Alt+Insert` e escolha o tipo de classe

## Templates Disponíveis

- `Controller.java.template` - Template para REST Controller
- `Service.java.template` - Template para Service Layer
- `Repository.java.template` - Template para Repository (JPA)
- `Entity.java.template` - Template para Entidade JPA
- `DTO.java.template` - Template para Data Transfer Object
